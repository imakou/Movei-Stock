## Movie Stock

A movie information website application as a side-project based on React and Redux.

[Demo](http://movie-stock.surge.sh/)

## Features:

- Mobile devices friendly
- Search movies
- Now Playing / Popular
- Movie's detail
- Basic Info
  - Media
    - Posters
    - Trailers
    - Images
    - Review
  - Casts
